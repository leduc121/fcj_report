### Kết quả đạt được tuần 2:

* **Kiến thức VPC:**
  * Hiểu kiến trúc VPC và cách thiết kế topology mạng trong AWS
  * Học được sự khác biệt giữa public và private subnets
  * Nắm vững khái niệm routing với Route Tables và Internet Gateway
  * Hiểu NAT Gateway cho truy cập internet từ private subnets
  * Học về bảo mật mạng với Security Groups và Network ACLs
  * Tạo thành công VPC với cấu hình subnet phù hợp

* **Kiến thức EC2:**
  * Hiểu được các loại EC2 instance và trường hợp sử dụng
  * Học về AMI và cách chọn image phù hợp
  * Hiểu về EBS volumes và các tùy chọn lưu trữ
  * Nắm vững các phương thức kết nối SSH đến EC2 instances
  * Học về Elastic IP cho địa chỉ IP tĩnh
  * Khởi chạy và quản lý thành công EC2 instances trong VPC

* **Phát triển React:**
  * Hiểu React cơ bản và kiến trúc dựa trên component
  * Học về cú pháp JSX và composition component
  * Nắm vững quản lý state với useState hook
  * Hiểu lifecycle component và useEffect hook
  * Xây dựng ứng dụng React đơn giản với functional components
  * Có nền tảng cho phát triển frontend hiện đại
